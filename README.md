# 🧮 Interactive Calculator Web App

An interactive calculator web application built using HTML, CSS, and JavaScript. This project performs basic arithmetic operations with a smooth and user-friendly interface, demonstrating core front-end development concepts.

## 🚀 Features

* Perform basic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (÷)
* Interactive button-based input
* Clear (reset) functionality
* Delete last input (backspace)
* Prevents multiple decimal inputs
* Error handling (e.g., division by zero)
* Continuous calculations support

Interactive calculators like this are commonly used as beginner projects to practice DOM manipulation and event handling in JavaScript ([DEV Community][1]).

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## ⚙️ How It Works

* User clicks numbers → displayed on screen
* Operator is selected → stored internally
* Second number is entered
* Clicking `=` performs calculation
* Result is displayed instantly

## ▶️ How to Run

   ```
   git clone https://github.com/rinshuDeveloper/interactive-calculator.git
   ```

2. Open the project folder

3. Run `index.html` in your browser
